## 1. Входная цепь USB-C

```
USB-C J1 (JAE DX07S024WJ3R400)
  VBUS ──── F1 (500мА PTC) ──── +5V_USB
  D+   ──── U_ESD ────────────── FT231XS PIN14 (D+)
  D-   ──── U_ESD ────────────── FT231XS PIN13 (D-)
  GND  ──── GND
  CC1/CC2 ── 5.1кОм ── GND     (детектирует устройство, не зарядник — 5V 500мА)
```

**Компоненты:**
- **J1**: JAE DX07S024WJ3R400 — USB-C вертикальный SMD, монтаж в центре платы
- **F1**: Bourns MF-MSMF050-2 — PTC 500мА самовосстанавливающийся предохранитель
- **U_ESD**: PRTR5V0U2X (SOT-363) — TVS на линии D+/D−, 3.3V клэмп
- **R_CC101, R_CC102**: 5.1 кОм 0402 × 2 — идентификация USB device (не Host/PD)

**Подключение пинов J1 (DX07S024WJ3R400):**

| Пины J1 | Сигнал | Куда |
|---|---|---|
| A4, A9, B4, B9 | VBUS | → F1 → +5V_USB |
| A1, A12, B1, B12 | GND | GND |
| 25, 26, 27, 28 | SHIELD | GND |
| A5 | CC1 | R_CC1 → GND |
| B5 | CC2 | R_CC2 → GND |
| A6, B6 | D+ | net D_P → U_ESD I/O2 |
| A7, B7 | D− | net D_N → U_ESD I/O1 |
| A8, B8 | SBU1/2 | No Connect |
| A2,A3,A10,A11,B2,B3,B10,B11 | SuperSpeed TX/RX | No Connect |

## 2. Шина +5V_A — LC-фильтр

BLM18 феррит имеет ~8 Ом на 250–500 кГц → недостаточно для AD8000.
Используем LC фильтр, идентичный −5V_A: 81 дБ подавления на 250 кГц.

```
+5V_USB ── L1/L101 (47мкГн) ──┬── +5V_A
                               ├── C3/C101 10мкФ тант. ── GND
                               └── C4/C102 100нФ X7R  ── GND
```

**Компоненты:**
- **L1 (KiCad: L101)**: Bourns SRN3015-470M — 47 мкГн, 400 мА, ±20%, DCR 1.4 Ом, экранированный 3.0×3.0×1.5 мм
- **C3 (KiCad: C101)**: 10 мкФ тант., 10В, 0805
- **C4 (KiCad: C102)**: 100 нФ X7R 0402

**Расчёт:**
- f_c = 1/(2π√(47мкГн × 10мкФ)) = **73 Гц**
- Ослабление на 300 кГц: (300000/73)² = **15600×** = **84 дБ**
- USB ripple 50 мВ → +5V_A ripple **< 0.003 мВ** → AD8000 вход < 0.2 мкВ ✓
- Просадка на DCR: 30 мА × 1.8 Ом = **54 мВ** → +5V_A = 4.946 В (AD8000 Vin_min = 4.5В ✓)

## 3. +3.3V_A — LP5907MFX-3.3 (аналоговое питание)

```
+5V_USB ─── C5/C103 1мкФ ─── [LP5907 IN] ─── [LP5907 OUT] ─── +3.3V_A
                                    │                  │
                                   GND       C6/C106 4.7мкФ ── GND
                                             C7/C108 100нФ  ── GND
```

**Компонент:** LP5907MFX-3.3/NOPB (SOT-23-5)
- Шум: **6.8 мкВ rms** (10Гц–100кГц) — лучший в классе для VDDA / TL431DBZ (U301)
- PSRR: 79 дБ @ 1 кГц
- Dropout: 90 мВ @ 150 мА → Vin_min = 3.39 В, от 5В — огромный запас
- Ток до 250 мА (нам нужно ~50 мА)

**Пины LP5907MFX-3.3 (SOT-23-5, DBV):**

| Пин | Имя | Куда |
|---|---|---|
| 1 | IN | +5V_USB + C5 → GND |
| 2 | GND | GND |
| 3 | EN | +5V_USB (pull high = всегда включён) |
| 4 | N/C | No Connect (**не NR** — NR пин отсутствует в SOT-23) |
| 5 | OUT | +3.3V_A + C6 ‖ C7 → GND |

**Компоненты обвязки:**
- **C5 (KiCad: C103)**: 1 мкФ X5R 0402, 10В (вход)
- **C6 (KiCad: C106)**: 4.7 мкФ X5R 0402, 10В (выход, параллельно C7)
- **C7 (KiCad: C108)**: 100 нФ X7R 0402 (ВЧ развязка, параллельно C6)

## 4. +3.3V_D — AMS1117-3.3 (цифровое питание)

```
+5V_USB ─── C8/C104 10мкФ тант. ─── [AMS1117 IN] ─── [AMS1117 OUT] ─── +3.3V_D
                                           │                   │
                                          GND        C9/C105 10мкФ тант. ── GND
                                                      C10/C107 100нФ X7R  ── GND
```

**Компонент:** AMS1117-3.3 (SOT-223)

> **KiCad символ:** `Regulator_Linear:AMS1117CD-3.3` (TO-252). Footprint `Package_TO_SOT_SMD:SOT-223-3_TabPin2` ✓ назначен.

**Пины AMS1117CD-3.3 (в KiCad символе):**

| Пин | Имя | Куда |
|---|---|---|
| 3 | VI | +5V_USB + C8 → GND |
| 2 | VO | +3.3V_D + C9 ‖ C10 → GND |
| 1 | GND | GND |

**⚠ Внимание:** AMS1117 требует ESR на выходе 0.01–1 Ом для устойчивости.  
С чисто керамическими конденсаторами — возможна генерация!  
→ Обязательно: **10 мкФ танталовый** на входе и выходе.

- Dropout: 1.1 В @ 800 мА → Vin_min = 4.4 В, от 5В — запас 600 мВ ✓
- Рассеиваемая мощность: (5 − 3.3) × 0.15 А = **0.26 Вт** → SOT-223 нормально (θja = 80°C/W → ΔT = 21°C ✓)

**Компоненты обвязки:**
- **C8 (KiCad: C104)**: 10 мкФ танталовый, 10В, 0805 (вход)
- **C9 (KiCad: C105)**: 10 мкФ танталовый, 10В, 0805 (выход — критично для стабильности!)
- **C10 (KiCad: C107)**: 100 нФ X7R 0402 (ВЧ развязка)

## 5. −5V_A — TPS60400 + LC-фильтр

### 5.1 Схема TPS60400

**Пины TPS60400DBV (SOT-23-5, KiCad символ):**

| Пин | Имя KiCad | Куда |
|---|---|---|
| 2 | IN | +5V_A + C12/C110 (100нФ bypass) → GND |
| 4 | GND | GND |
| 5 | CFLY+ | один конец C11/C111 (летающий конд.) |
| 3 | CFLY− | другой конец C11/C111 (летающий конд.) |
| 1 | OUT | C13/C112 (10мкФ тант.) → GND; далее → L2/L102 → −5V_A |

```
+5V_A ─── IN(2) [TPS60400] OUT(1) ─── C13/C112 10мкФ тант. ─── GND
                 CFLY+(5) ── C11/C111 100нФ ── CFLY−(3)
                 GND(4) ─── GND
OUT(1) ─── L2/L102 47мкГн ─── −5V_A
```

**Компонент:** TPS60400DBVT (SOT-23-5)
- C11 **(KiCad: C111)** (летающий): **100 нФ X7R 0402** между CFLY+ и CFLY− (критично! X7R, не Y5V)
- C12 **(KiCad: C110)** (bypass VIN): **100 нФ X7R 0402**
- C13 **(KiCad: C112)** (выходной): **10 мкФ танталовый**, полярность обратная (+ к GND)
- f_sw = 250 кГц

### 5.2 LC-фильтр на выходе −5V

**Почему LC, а не феррит:**  
BLM18 феррит имеет сопротивление ~5 Ом на 250 кГц → слабое ослабление.  
LC-фильтр даёт:  f_c = 1/(2π√LC) = 73 Гц → ослабление на 250 кГц = **81 дБ**.

```
−VOUT_raw ── L2/L102 47мкГн ──┬── −5V_A
                               ├── C14/C113 10мкФ тант. ── GND
                               └── C15/C114 100нФ X7R  ── GND
```

**Расчёт:**
- L2/L102 = 47 мкГн, C14/C113 = 10 мкФ → f_c = 73 Гц
- На 250 кГц: затухание = (250000/73)² = **11740×** = **81 дБ**
- Пульсации 25 мВ → после фильтра: **< 0.01 мВ** на −5V_A
- AD8000 PSRR@250кГц = 35 дБ → на вход < 0.3 мкВ → пренебрежимо ✓

**Компоненты:**
- **L2 (KiCad: L102)**: Bourns SRN3015-470M — 47 мкГн, 400 мА, экран, 3.0×3.0×1.5 мм, footprint Neosid SMS-ME3015
- **C14 (KiCad: C113)**: 10 мкФ тант., 16В, 0805 — **полярность обратная** (+ к GND, − к −5V_A)
- **C15 (KiCad: C114)**: 100 нФ X7R 0402

> ⚠ **Полярность на −5V:** C13/C112 и C14/C113 — танталовые конденсаторы на отрицательной шине. Плюс подключается к GND (0В), минус к −5В. На схеме и при пайке обязательно проверить.

## 6. Шина +5V_HV — фильтр + транзит на плату ВВ

Шина **+5V_HV** передаёт отфильтрованный +5V_USB на плату ВВ через header.  
Фильтр размещён здесь (а не на HV board) — чтобы помехи от MAX1847 @ 252 кГц не попадали обратно в +5V_USB шину.

```
+5V_USB ── L3/L103 100мкГн ──┬── pin 24 header (+5V_HV) ──→ HV board [MAX1847 VIN]
                              ├── C19/C115 47мкФ X5R ── GND
                              └── C20/C116 100нФ X7R ── GND
```

**Компоненты:**
- **L3 (KiCad: L103)**: Bourns SRU5028-101Y — 100 мкГн, 470 мА, экран, DCR 625 мОм, 5.2×5.2×2.8 мм
- **C19 (KiCad: C115)**: 47 мкФ X7R 10В, 1210 (GRM32ER71A476KE15L, Murata — общий с C206 на HV board); при DC 5В эффективная ёмкость ~33 мкФ
- **C20 (KiCad: C116)**: 100 нФ X7R 0402

**Расчёт подавления @ 252 кГц:**
- f_c = 1/(2π√(100мкГн × 33мкФ)) ≈ **88 Гц**
- Затухание на 252 кГц: (252000/88)² ≈ **8.2×10⁶** = **138 дБ**

**Бюджет тока +5V_HV:**  
MAX1847 + IRLML0040 + CW-умножитель: **~112 мА** @ 5В (typ при U_HV = 1200 В, I_det = 50 нА)  
Просадка на L3 (DCR ≈ 2 Ом): 112 мА × 2 Ом = **224 мВ** → +5V_HV = 4.78 В (MAX1847 Vin_min = 4.5 В ✓)

## 8. Индикация (LED D102/D2, D101/D5)

**D2 — статус (зелёный), управление от MCU:**
```
global label LED ── R2/R102 (330Ом) ── D2/D102 anode ── D2/D102 cathode ── GND
```
- Активный высокий (MCU HIGH → LED светится)
- Состояния кодируются паттернами мигания в прошивке
- Управляющий сигнал: global label `LED` → MCU GPIO PB6

| LED | Цвет | Сигнал | MPN (0402) |
|---|---|---|---|
| D2 (KiCad: D102) | Зелёный | global label LED | Würth 150040GS73240 |
| D5 (KiCad: D101) | Синий | FT231XQ CBUS0 | Würth 150040BS73240 |

**D5 — USB активность (синий), от FT231XQ CBUS0:**
```
FT231XQ CBUS0 (пин 15) ── R1/R101 (330Ом) ── D5/D101 anode ── D5/D101 cathode ── GND
```
- Не требует прошивки, управляется FT231XQ аппаратно
- Настройка через FT_PROG: CBUS0 = TXLED

Резисторы R1/R101, R2/R102: 330 Ом 0402 (ток ~10 мА @ 3.3В).

## 10. Ключевые проверки перед разводкой

- [ ] C105 (AMS1117 выход, old C9) — танталовый, не керамический
- [ ] C111(CFLY), C110(VIN) (TPS60400 насос, old C11/C12) — X7R, не Y5V/Z5U
- [ ] L101, L102 (47 мкГн, old L1/L2) — ток насыщения > 100 мА (SRN3015-470M: 400 мА ✓)
- [ ] R_CC101/R_CC102 = 5.1 кОм — иначе USB-C хост не даст 5В
- [ ] +3.3V_A и +3.3V_D — **раздельные трассы** от разъёма J2 до потребителей
- [ ] −5V_A трасса — короткая, конденсаторы LC фильтра рядом с TPS60400
- [ ] Pin 24 header = **+5V_HV** → через L103+C115+C116 фильтр (не напрямую с +5V_USB)
- [ ] L103 (100мкГн, old L3) — ток насыщения > 200 мА (SRU5028-101Y: 470 мА ✓)
