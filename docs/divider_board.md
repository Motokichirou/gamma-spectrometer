# Делитель ФЭУ для Hamamatsu R1306/R1307

## Назначение
Активный делитель напряжения для гамма-спектрометрии с NaI(Tl) кристаллом
(типично 3"×3"), оптимизированный для скоростей счёта до 30 кимп/с.

## Содержимое проекта

- `pcb/gamma_spectrometer/gamma_spectrometer.kicad_pro` — единый KiCad проект (все 4 платы)
- `pcb/gamma_spectrometer/divider_board.kicad_sch` — схема платы делителя (в KiCad ✓, ERC ✓)
- `pcb/gamma_spectrometer.net` — нетлист (авторитетный источник, после Export Netlist)

| mid_K - Dy1/G | 1.0 | R402 = 750 кОм | 145 В | 1206 |
| Dy1 - Dy2 | 1.0 | R403 = 680 кОм | 132 В | 1206 |
| Dy2 - Dy3 | 1.0 | R404 = 680 кОм | 132 В | 1206 |
| Dy3 - Dy4 | 1.0 | R405 = 680 кОм | 132 В | 1206 |
| Dy4 - Dy5 | 1.0 | R406 = 680 кОм | 132 В | 1206 |
| Dy5 - Dy6 | 1.0 | R407 = 680 кОм | 132 В | 1206 |
| Dy6 - Dy7 | 1.2 | R409 = 820 кОм | 159 В | 1206 |
| Dy7 - Dy8 | 1.4 | R410 = 1.0 МОм | 194 В | 1206 |
| Dy8 - GND | 1.0 (×0.75) | R412 = 1.0 МОм | 194 В | 1206 |

⚠ **Фикс C1 (2026-06-15):** низ цепочки Dy8→GND теперь один **R412 = 1.0 МОм** (194В, 1206); прежние R412/R414 (2×499к) и узел mid_P **убраны**. Коллектор буфера Q403 переведён с mid_P на **жёсткую GND** (см. ниже) → средний динодный ток идёт мимо цепочки: просадка усиления **−4.7% → −0.14%** (T_C1 PASS, REVIEW_round3).
⚠ Низ цепочки (R412) уходит на **GND напрямую**, минуя анод P — ток делителя не течёт через сигнальный узел.
⚠ **R1 (watch):** Vce(Q403) после фикса = **195 В** (коллектор на GND, эмиттер Dy8 ≈ −195 В); Vceo MMBTA42 = 300 В → запас 105 В, в норме, но учитывать при ВВ-транзиентах/дерейтинге.
⚠ **R2 (КРИТИЧНО для разводки — звёздная земля):** коллектор Q403 теперь сажает самый большой импульсный динодный ток (пик Dy8 ~0.7 мА) на **общую GND**, куда возвращается и анодный сигнал (Rin/R415). Ток коллектора Q403 вести **отдельной трассой к точке-звезде GND**, подальше от сигнального возврата анода — иначе импульс динода через общий импеданс земли подмешается в сигнал (gain/линейность). Проверяется только на плате (REVIEW_round3 R2).

Сумма множителей: 11.1, базовое деление: 1500/11.1 = 135 В.

Фокусирующая сетка G подключена к узлу между R401 и R402 (mid_K).
Это создаёт соотношение K-G : G-Dy1 = 1:1 (750кОм : 750кОм),
что соответствует таблице распределения напряжений из даташита R1307 (Note 1).
G — отдельный электрод, НЕ объединяется с Dy1.

## Активные буферы

Эмиттерные повторители на NPN-транзисторах MMBTA42 (Vce=300В, SOT-23)
установлены на трёх последних динодах: Dy6, Dy7, Dy8.

Принцип работы: транзистор берёт пиковый ток нагрузки динода со стороны
коллектора (через резистор делителя) и подаёт его на эмиттер (вывод динода),
не нагружая опорное напряжение в межимпульсные интервалы. Накопительный
конденсатор между эмиттером и соседним (более отрицательным, в сторону катода)
динодом обеспечивает мгновенный ток при импульсе.

| Динод | Транзистор (KiCad) | База-резистор | Накопит. C |
|---|---|---|---|
| Dy6 | Q401 (MMBTA42) | R408 = 10к | C408 = 22 нФ / 1 кВ |
| Dy7 | Q402 (MMBTA42) | R411 = 10к | C409 = 47 нФ / 1 кВ |
| Dy8 | Q403 (MMBTA42) | R413 = 10к | C410 = 100 нФ / 1 кВ |

**⚠ Топология (критично, исправлено по аудиту нетлиста 2026-06-11):**
- Узел цепочки (между резисторами) → базовый резистор → **база**
- **Эмиттер** → вывод динода ФЭУ. Динод видит ТОЛЬКО эмиттер, цепочка динода НЕ касается!
- Накопительные конденсаторы — между соседними динодными (эмиттерными) линиями
- Коллектор — Q401.C→DY7, Q402.C→DY8; **Q403.C→GND** (фикс C1: жёсткая шина, не mid_P)
- Динод сидит на V_узла − V_BE (≈0.65 В) — на фоне 130–190 В/ступень несущественно
- Если цепочку соединить с динодом напрямую — база через R окажется на собственном эмиттере, V_BE = 0 навсегда, транзистор заперт, буфер мёртв (так и было до исправления)

Контрольные цепи (по нетлисту):
```
Узел DY6: R407.2 + R409.1 + R408.2 + C407.2      Динод DY6: Q401.E + пин 8 + C408.2 + C409.1
Узел DY7: R409.2 + R410.1 + R411.2               Динод DY7: Q402.E + пин 9 + C409.2 + C410.1 + Q401.C
Узел DY8: R410.2 + R412.1 + R413.2               Динод DY8: Q403.E + пин 10 + C410.2 + Q402.C
```

Просадка напряжения на накопительных конденсаторах (расчёт для старого 12 нКл; реальный заряд ~1.14 нКл ⇒ просадка ещё ~×10 меньше):
- C408 (22 нФ): 0.55 мВ → 0.4% от 136 В (Dy6 уровень)
- C409 (47 нФ): 0.26 мВ → 0.16%
- C410 (100 нФ): 0.12 мВ → 0.06%

## Анодная цепь и предусилитель AD8000

Анод (P, pin 11) подключается **напрямую** к Rin и Rg. **C11 не используется** — в цепочке делителя P итак близок к 0В, DC-разделение не требуется и вредит сигналу.

С узла **P (анод)**:
- **Rin = 49.9 Ом E96** (R415, 1%) — к AGND. Нагрузка анода и путь возврата сигнального тока.
- **Rg = 100 Ом** (R416, 1%) — к VINM AD8000 (инвертирующий вход).

Эффективное сопротивление нагрузки анода: Rin ‖ Rg = 49.9‖100 = **33.3 Ом**  
→ V_anode (Cs-137) = 0.94 мА × 33.3 Ом = **31.3 мВ** ✓  
DC потенциал анода ≈ **0 мВ** (ток делителя возвращается на GND через R412, минуя анод).

⚠ **НЕ ставить GND-символ на P.** Rin — единственный путь возврата сигнального тока анода к AGND. Если поставить жёсткий GND-символ на P, весь сигнальный ток уйдёт в него, на VINM AD8000 сигнала не будет.


### AD8000 — инвертирующий усилитель ×(−15)

Верифицирован в sim1 (sim1_ad8000.cir): BW = 99 МГц, peaking < 0.5 дБ.

```
P (анод) ──── Rin(50Ом) ──── AGND
         │
         └─── Rg(100Ом) ──┬── −IN (pin 2)
                          │
       FEEDBACK (pin 1) ──┴── Rfb 1.5кОм ──┘   ‖ Cfb 1пФ NP0
       (Rfb/Cfb между FB и −IN!)

              +IN (pin 3) = AGND
              PD  (pin 8) = +5V_A
              −VS (pin 4) = −5V_A
              +VS (pin 7) = +5V_A
              EP  (exposed paddle) = AGND

              OUTPUT (pin 6) ──── VOUT → U.FL → MCU (нагрузка, БЕЗ Rfb)

⚠ FEEDBACK(1) внутри = OUTPUT(6) (даташит «Low Distortion Pinout»). Поэтому Rfb
обязан идти на −IN, а НЕ между OUT и FB (см. cheatsheet #11). gain = −Rfb/Rg = −15.
```

**Схема включения:**
```
+5V_A ─┬─ 4.7мкФ X5R 0805 ─ AGND    → AD8000 +VS (pin 7)
        └─ 100нФ X7R  0402 ─ AGND

-5V_A ─┬─ 4.7мкФ X5R 0805 ─ AGND    → AD8000 −VS (pin 4)
        └─ 100нФ X7R  0402 ─ AGND
```
*(Феррит-бусины BLM18 не используются: ±5V_A уже отфильтрованы LC-фильтрами
 на USB/Power board — 81–84 дБ @ 250 кГц. Локальные развязочные конденсаторы
 непосредственно у пинов питания AD8000 достаточны.)*

AD8000 PD  (pin 8) → +5V_A (усилитель включён: Enable > +VS − 1.9 В)
AD8000 +IN (pin 3) → AGND
AD8000 EP  (exposed paddle) → AGND
AD8000 NC  (pin 5) → не подключать
AD8000 FEEDBACK (pin 1) → Rfb(1.5к)‖Cfb(1пФ) → −IN (pin 2); НЕ на GND/потенциал
AD8000 OUTPUT (pin 6) → U.FL разъём J_sig + шейпер (нагрузка; Rfb сюда НЕ подключать)
```

**Компоненты AD8000:**

#### Опция (v2): вход как TIA на ADA4817 — НЕ для v1

Альтернатива AD8000-инвертору: **ADA4817-1 в схеме трансимпеданса (TIA)**. Подтверждено
симуляцией (`simulation/sim_tia_real.cir`, официальная модель ADI + `sim_tia.cir` поведенческая,
сошлись): **σ_total 94.4 → 76.9 мкВ (−18%)**, σ_baseline 0.31 → 0.25 кан.; анод строго на 0 В
(идеальная линейность). Причина: TIA noise gain ≈1 vs ×11 у инвертора + FET-вход (i_n=2.5 фА).
**НО на разрешение v1 не влияет** — упираемся в статистику кристалла (~6% FWHM ≈ 68 кан.),
электронный шум ~0.6 кан. пренебрежим. Это запас на лучший кристалл/ФЭУ, не починка.
**Решение: v1 = AD8000; ADA4817-TIA — кандидат на v2.**

ADA4817-1 (SOIC-8) — **корпус RD-8-4 и распиновка ИДЕНТИЧНЫ AD8000** (оба ADI low-distortion:
1=FB, 2=−IN, 3=+IN, 4=−Vs, 5=NIC, 6=OUT, 7=+Vs, 8=PD) → по плате прямой drop-in, меняется обвязка:

| Действие | Компонент | Было (AD8000) | Стало (TIA ADA4817) |
|---|---|---|---|
| ЗАМЕНА | U401 | AD8000YRDZ | **ADA4817-1ARDZ** (тот же SOIC RD-8-4) |
| **УБРАТЬ** | R415 (Rin) | 49.9 Ом → AGND | — (TIA: анод прямо на −IN, виртуальная земля) |
| **УБРАТЬ** | R416 (Rg) | 100 Ом → −IN | — (ток ФЭУ втекает прямо в −IN) |
| ИЗМЕНИТЬ | R417 (Rfb) | 1.5 кОм | **499 Ом** (трансимпеданс = Rf → 3.5 МэВ ≈ 2.3 В как сейчас) |
| ИЗМЕНИТЬ | C415 (Cfb) | 1 пФ | **~3 пФ** (устойчивость TIA при Cin≈10 пФ) |
| БЕЗ ИЗМ. | C411–C414 | развязка ±5V_A | то же |

Топология: анод P → −IN(2) напрямую; Rf(499)‖Cf(3п) между FB(1) и −IN(2); +IN(3)→AGND;
OUT(6)→U.FL/шейпер; PD(8)→+5V_A.

⚠ Нюансы v2: анод на виртуальной земле (0 В), Rin-возврат не нужен (DC анода возвращается через Rf),
правило «не ставить GND на P» неактуально (P=−IN≈0 В); Cin (анод+монтаж+1.3 пФ ОУ ≈10 пФ) держать
минимальной (устойчивость/шум); ±5В = 10 В у ADA4817 близко к max (abs 10.6 В) — проверить ±5V_A.

**Ключевые требования к разводке (из sim1):**
- C_par на VINM ≤ **1 пФ** → трасса от Rg/Rfb к VINM: минимальная длина, без via
- C_fb = 1 пФ **обязателен**: компенсирует C_par, устраняет peaking
- Развязки ±5V_A: ≤ 5 мм от пинов V+/V−
- AGND: отдельный полигон в зоне AD8000, одна точка соединения с GND делителя

## Развязочные конденсаторы

Параллельно каждому резистору цепочки (кроме R409, R410, R412, которые
шунтированы накопительными C408-C410 или работают с малым напряжением)
установлен высокочастотный развязочный конденсатор 100 нФ X7R:
- C401 (на R401, 145В): 200 В рейтинг
- C402 (на R402, 145В): 200 В рейтинг
- C403-C407 (на R403-R407, до 132В): 200 В рейтинг

R407 (Dy5→Dy6), R409 (Dy6→Dy7), R410 (Dy7→Dy8) — отдельные bypass не нужны: каждый из них
перекрыт накопительным конденсатором буфера (C408, C409, C410 соответственно через эмиттеры Q401/Q402/Q403).
R412 (Dy8→GND, 1.0 МОм, 194В) — нижний резистор к жёсткой GND, bypass не обязателен (при желании 100нФ 250В).

Развязки снижают динамический импеданс делителя на ВЧ и подавляют
наводки от соседних импульсов.

## Подключение к остальным платам стека

### 1. −ВВ от платы ВВ питания
**TP1** (-HV): экранированный провод ≥ 5 кВ изоляция (ПВВ-1 или аналог).
Приходит от платы ВВ, минуя плату МК (вдоль края стека).
Минимальный зазор от других цепей: 5 мм.

### 2. GND
**TP2**: провод возврата, соединяется с GND стека на плате МК.

### 3. Питание AD8000 (от платы МК)
**J_pwr** — 3-пиновый JST PH 2.0 мм:

| Пин | Сигнал | От платы МК |
|---|---|---|
| 1 | +5V_A | pin 1 header MCU→divider |
| 2 | AGND | GND |
| 3 | −5V_A | pin 2 header MCU→divider |

Кабель: витая пара (±5V_A) + GND, длина ≤ 50 мм.

### 4. Сигнал AD8000 (на плату МК)
**J_sig** — U.FL разъём. Выход AD8000 (±431 мВ для Cs-137, верифицировано sim2).
Кабель: RG-178 (50 Ом), длина ≤ 100 мм, на плате МК → U.FL вход шейпера.

*Коаксиал теперь несёт УСИЛЕННЫЙ сигнал ±431 мВ (Cs-137) вместо прежних 31 мВ —
на порядок меньше чувствителен к наводкам.*

## Pt1000 термодатчик

Pt1000 (Honeywell 700-102BAA-B00, 1 кΩ @ 0°C, M222 leaded) **не размещается** на
плате делителя. Вместо этого он крепится снаружи на корпус кристалла
(тонкая медная скобка или термоклей) и подключается к плате АЦП витой
парой. Это даёт лучшую корреляцию с температурой кристалла, который
является основным источником температурного дрейфа (-0.3%/°C световыхода
для NaI(Tl)).

## Рекомендации по разводке PCB

1. Плата круглая, диаметр **~42-45 мм** (увеличено с 35-40 мм для размещения AD8000).
2. Двусторонняя плата, FR-4 толщиной 1.6 мм. Класс точности 5 или выше.
3. **Зазоры по высокому напряжению** (правило: 1 мм на 500 В для FR-4):
   - Между трассой -HV и любой соседней трассой/полигоном: ≥ 3 мм
   - Между Dy1 и анодом: ≥ 2 мм (макс. разница ~1500 В)
   - Между соседними динодами: ≥ 0.5 мм (разница ~136-200 В)
4. **GND-полигон**: сплошной AGND в зоне AD8000 + анодной цепи.
   Обходит зону высокого напряжения с зазором 3 мм.
   AGND соединяется с GND делителя в одной точке (звёздная земля).
5. **Зона AD8000** — противоположная сторона платы от HV цепочки:
   - AD8000 (SOIC-8): как можно ближе к анодному выводу ФЭУ (P, J_pmt pin 11)
   - Rfb + Cfb: ≤ 2 мм от VINM пина AD8000
   - C_par на VINM: ≤ 1 пФ (трасса от Rg к VINM — минимальна, без via)
   - Развязочные конденсаторы ±5V: ≤ 5 мм от V+/V− пинов AD8000
6. **U.FL разъём J_sig**: у края платы, рядом с AD8000 VOUT
7. **JST PH J_pwr** (3 пина): у края платы, в зоне аналогового питания
8. **Транзисторы Q401-Q403 и накопительные конденсаторы C408-C410**: близко к динодным выводам.
9. После сборки: покрыть лаком зону ВВ-резисторов и анодную цепь.

## Важные замечания

- **Никогда не подавать ВВ без подключенного делителя** — без нагрузки
  ВВ-БП может выйти на превышение опорного напряжения.
- **Порядок включения**: сначала USB, проверить мониторинг ВВ через
  ПО → плавно поднять ВВ → начать измерения.
- **Порядок выключения**: выключить ВВ через ПО, дождаться разряда
  накопительных конденсаторов (5-10 секунд), затем отключать USB.
- **Пайка коаксиала (на плате МК)**: лудить только центральную жилу, оплётку зачищать
  и припаивать к GND-площадке U.FL максимально коротко для минимизации индуктивности.
