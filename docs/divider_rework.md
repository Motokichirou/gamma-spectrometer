## 🔴 1. КРИТИЧНО: буферы Q401-Q403 НЕ смещены (подтверждено симуляцией) — ✅ ИСПРАВЛЕНО (см. статус выше)

**Факт (sim `simulation/sim_buffer_bias.cir`, топология 1:1 из нетлиста):**
| Транзистор | Ic квази | Должно (хэндбук) |
|---|---|---|
| Q401 | **0.19 нА** | ~194 мкА |
| Q402 | **0.19 нА** | ~194 мкА |
| Q403 | ~0 | ~194 мкА |

Транзисторы в **отсечке** (на ~6 порядков ниже рабочего тока). Диноды DY6/7/8 плавают (напряжение задаётся утечками, не жёстким источником).

**Корень:** ток делителя пущен по **параллельной ветке** (узлы NodeX=`Net-(C407-Pad2)` → NodeY → NodeZ → GND, резисторы R407/R409/R410/R412), **мимо транзисторов**. Диноды висят на эмиттерах без DC-возврата. Там, где у хэндбука **смещающий резистор эмиттер→предыдущий динод** (по нему течёт ток делителя и держит транзистор открытым), у нас стоит **конденсатор (C408)**.

**Последствие:** не работающий активный делитель — транзисторы открываются из отсечки только под импульсом → задержка, нелинейность, crossover на малых сигналах; диноды без жёсткого DC.

**Фикс (по Fig 5-15):** вести цепочку делителя **ЧЕРЕЗ диноды** — эмиттерные резисторы DY5-DY6, DY6-DY7, DY7-DY8, по которым идёт ток делителя (≈194 мкА) и смещает транзисторы; базы — через свои R от узлов цепочки; **убрать параллельную ветку NodeX/Y/Z**.

## ⚠ 2. Конденсаторы — пересобрать
- **Убрать C402-C407** (G, Dy1-Dy5): импульсный ток ранних динодов ≪ тока цепочки (Dy5 ~16 мкА vs 194 мкА) — bypass не нужен (sim/расчёт; референс их не ставит).
- **Bypass только на Dy6-Dy8** (последние ступени): по E-C каждого транзистора.
- **Q403 (E=DY8, C=GND): добавить недостающий E-C кап (DY8↔GND)** — последний, самый сильноточный каскад.
- C408 (DY5-DY6) в текущем виде — на месте смещающего резистора; решится п.1.
