# Обзор аппаратной части

<!-- README-I18N:START -->
**Русский** | [English](../en/hardware.md)
<!-- README-I18N:END -->

Прибор — цилиндрический стек из четырёх плат Ø45 мм, установленных прямо за
сцинтиллятором NaI(Tl) Ø63×63 мм, сопряжённым с ФЭУ Hamamatsu R1307. Единственный
кабель USB-C выходит с торца и несёт и питание, и данные.

## Стек плат (снизу → вверх)

```
[торец — кабель USB-C]
  1. USB/Power board   Ø45 мм   USB-C (JAE DX07S024WJ3R400)
  2. HV board          Ø45 мм   MAX1847 flyback + 8-ступ. CW умножитель, 0…−1500 В
  3. MCU board         Ø45 мм   STM32G474, шейпер CR-RC², ADC
  4. Divider board     Ø42–45 мм  делитель R1307 + ADA4817 TIA
[торец — ФЭУ R1307 + NaI(Tl)]
```

### Межплатные соединения

| Связь | Откуда → Куда | Разъём |
|---|---|---|
| Питание/управление стека | плата ↔ плата | pin-header 2×N 2.54 мм |
| Сигнал | делитель → МК | коаксиал RG-178 + U.FL |
| Аналоговое питание делителя | МК → делитель | JST PH 3-пин (+5V_A, AGND, −5V_A) |
| −ВВ | ВВ → делитель | экранированный провод ≥5 кВ вдоль края стека |

## Цепь сигнала

```
NaI(Tl)+R1307 → ADA4817 TIA (Zt=499Ω) → коакс → CR(OPAMP1) → RC(OPAMP2) → PGA×4(OPAMP3) → ADC
[плата делителя] ±5V_A         U.FL   τ=1 мкс       τ=510 нс      DC=2.13 В       3.3 В
V(анода)=31 мВ   amp_out=−431 мВ      [плата МК — G474]
```

**Полярность:** импульс на АЦП идёт *вниз* от baseline 2.13 В. В DSP:
`amplitude = baseline − sample`.

## 1. Плата USB/питания

Интерфейс USB-C и все шины питания стека.

- **FT231XQ** (QFN-20) — мост USB ↔ UART на 600 кбит/с, виден как виртуальный COM-порт.
- **LP5907** → +3.3V_A (малошумящий аналог), **AMS1117CD-3.3** → +3.3V_D (цифра).
- Зарядовый насос **TPS60400** → −5V_A.
- LC-фильтры 47 мкГн на +5V_A и −5V_A (≈84 дБ @ 300 кГц).
- **+5V_HV** — сырой USB 5 В через LC-фильтр на плату ВВ.
- ESD-защита **PRTR5V0U2X** на линиях данных USB.
- Бюджет USB 2.0 ≈ 370 мА < 500 мА.

Детальная схема: [`docs/usb_power_board_schematic.md`](../usb_power_board_schematic.md).

## 2. Плата ВВ

Генерирует регулируемое отрицательное высокое напряжение для ФЭУ.

- Контроллер flyback **MAX1847** + ключ **IRLML0040** + трансформатор **ATB322524** (1:10.2).
- 8-ступенчатый умножитель Кокрофта–Уолтона (CW), 0 … −1500 В.
- Питается от сырой шины +5V_HV (не от чистых аналоговых шин).
- Выходной RC-фильтр (3×100 кОм + ВВ-конденсаторы) для подавления пульсаций.
- Обратная связь: выход CW → делитель 1 ГОм → усилитель ошибки MAX1847; `V_HV = −V_CTRL × 454.5`.
- Мониторинг: делитель 68 МОм + опора **TL431** → `HV_MONITOR` на пин АЦП МК.
- `HV_ENABLE` активный HIGH (HIGH от МК = ВВ включено).
- Собранная плата заливается силиконом после проверки.

Детальная схема: [`docs/hv_board_schematic.md`](../hv_board_schematic.md).

## 3. Плата МК (v1.6)

Ядро обработки сигнала и интерфейса.

- **STM32G474MET6** @ 170 МГц со встроенными **OPAMP1/2/3** и 12-бит АЦП на 4 Мвыб/с.
- Вход: разъём U.FL с ~470 мВ @Cs-137 от ADA4817 TIA на плате делителя.
- Аналоговый шейпер: **CR** (10 кОм + 100 пФ + 10 кОм) → **RC** (1 кОм + 510 пФ + 1 кОм),
  формирует импульс типа CR-RC², далее PGA ×4 в АЦП.
- Опора напряжения **TL431** (2.495 В) плюс делители DC-смещения для baseline АЦП.
- SPI-флеш **W25Q64**, кварц 16 МГц.
- Датчик температуры **Pt1000** (Honeywell 700-102BAA-B00) на корпусе кристалла,
  читается ratiometric для термокомпенсации дрейфа.
- `HV_CONTROL` PWM → RC-фильтр → плата ВВ.

Детальный док: [`docs/adc_mcu_board.md`](../adc_mcu_board.md).

## 4. Плата делителя

Активный делитель ФЭУ и входной усилитель, ближе всего к трубке.

- **Hamamatsu R1307-01** (гибкие выводы), 11 пинов.
- Резистивная цепочка R401–R412 (тейпер под спектрометрию); bypass/накопительные капы
  только на последних динодах (C408/C409/C410 = 22/47/100 нФ на Dy6–Dy8).
- Три транзисторных буфера **MMBTA42** (Q401/Q402/Q403) на динодах Dy6/Dy7/Dy8, смещены
  током делителя (цепочка идёт через диноды эмиттерными резисторами, по Hamamatsu Fig 5-15).
  Защитные диоды B–E D401–D403 (1N4148W) — от обратного пробоя на рампе ВВ.
- **ADA4817-1ARDZ** трансимпедансный усилитель (TIA): анод напрямую на −IN (виртуальная
  земля), `Rf = 499 Ом`, `Cf = 4.7 пФ` (NP0), Zt = 499 Ом, полоса ≈ 92 МГц. На ~18% тише
  прежнего AD8000-инвертора.
- Питается от ±5V_A, поданных по стеку.

Детальный док: [`docs/divider_board.md`](../divider_board.md).
