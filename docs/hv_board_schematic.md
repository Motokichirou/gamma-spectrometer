# Плата ВВ питания — схемотехника

## Назначение
Средняя плата стека. Генерирует регулируемое высокое напряжение 0 … −1500 В
для ФЭУ R1307. Управляется сигналом HV_CONTROL от платы МК.
Питается напрямую от +5В USB через шину +5V_HV нижнего header'а.

## Стек и межплатные соединения

```
┌─ Плата МК (сверху) ──────────────────────────────────────────┐
│  Верхний 2×N header (размер при разводке)                      │
│  HV_CONTROL → (с платы МК вниз к нам)                         │
│  HV_MONITOR ← (от нас вверх к плате МК)                       │
└────────────────────────────────────────────────────────────────┘
            ↕ верхний header 2×N
┌─ Плата ВВ (эта плата) ───────────────────────────────────────┐
│  MAX1847 → IRLML0040 → ATB322524 → ×8 CW → −HV               │
│                              │                                 │
│                        шиль. кабель → плата делителя           │
└────────────────────────────────────────────────────────────────┘
            ↕ нижний header 2×N
┌─ USB/питание (снизу) ────────────────────────────────────────┐
│  +5V_HV — силовой, ≤ 200 мА                                    │
│  +5V_A, +3.3V_A, +3.3V_D, −5V_A, GND, UART, LED, SWD, ...   │
└────────────────────────────────────────────────────────────────┘
```

## 1. Питание +5V_HV

Приходит с USB/Power board через шину **+5V_HV** нижнего header'а.  
Bulk-фильтр (L103 100мкГн + C115 47мкФ) **находится на USB/Power board** — перед header'ом, чтобы помехи MAX1847 @ 252 кГц не попадали в +5V_USB шину.

Локальный bypass на HV board:
```
+5V_HV ─┬─ C204 100нФ X7R ── GND   (у MAX1847 IN pin 15)
         ├─ C206 47мкФ X5R  ── GND   (bulk, у T201 pin 1)
         └─ C207 100нФ X7R ── GND   (ВЧ развязка, у T201 pin 1)
```
C206+C207 стоят прямо перед pin 1 T201 — компенсируют паразитную индуктивность header'а при пиковых токах переключения.

Локальный LDO (+3.3V_local) **отсутствует** — убран за ненадобностью. TL431 (Vref_2.5В) питается напрямую от +5V_HV.

## 2. MAX1847EEE+ — контроллер ВВ

**Корпус:** QSOP-16

| Пин | Функция | Подключение |
|---|---|---|
| IN (15) | Вход питания | +5V_HV |
| VL (2) | Внутренний LDO ~4.25В | Развязка C202 0.47мкФ к GND; POL соединён с VL |
| PGND (12) | Силовая земля | GND |
| GND (10,11) | Сигнальная земля | GND |
| GATE / EXT (14) | Затвор MOSFET | → R206 10Ом → Q201 IRLML0040 Gate |
| CS (13) | Датчик тока | узел Source Q201; R205 (0.1Ом) к GND; параллельно C205 100пФ X7R |
| FREQ (3) | Задание частоты | R202 180кОм к GND |
| COMP (4) | Компенсация петли | R201 1МОм → C201 33нФ → GND (последовательно) |
| REF (5) | Опорное 1.25В | Развязка C203 0.1мкФ к GND (вых., не нагружать) |
| FB (6) | Обратная связь | ← R207/R209–R217 (10×100МОм от CW_out) + R208 (2.2МОм от HV_CONTROL) |
| SHDN (8) | Отключение (LOW=выкл) | ← R204 10кОм (от MCU HV_ENABLE); R203 100кОм к GND |
| POL (1) | Полярность N-ch | → VL (pin 2) |
| SYNC (16) | Синхронизация | GND (не используется) |

**Частота переключения:**

Задаётся только резистором R202 по формуле даташита:
```
T = 5.21×10⁻⁷ + 1.92×10⁻¹¹ × R202 − 4.86×10⁻¹⁹ × R202²   (R202 в Омах)
f_sw = 1 / T
```
При R202 = 180 кОм (180000 Ом):  T ≈ 3.96 мкс → **f_sw ≈ 252 кГц**

**HV_ENABLE (прямое подключение от MCU, активный HIGH):**
```
MCU HV_ENABLE (PB10) ── R204 10кОм ── SHDN (pin 8)
                                            │
                                        R203 100кОм
                                            │
                                           GND
```
- MCU HV_ENABLE = **HIGH (3.3В)** → SHDN = HIGH → MAX1847 работает → HV включён
- MCU HV_ENABLE = LOW / floating → SHDN подтянут к GND через R203 → MAX1847 остановлен → **HV = 0В**

Безопасное состояние по умолчанию: при сбросе или отсутствии команды MCU — HV выключено.
R204 (10кОм) защищает GPIO MCU от помех с HV-платы. При V_SHDN = 3.3В > V_IH ≈ 3В (0.7×VL) — порог надёжно пройден.

## 3. Первичный контур: IRLML0040 + ATB322524

### IRLML0040 — силовой ключ
- N-MOSFET, SOT-23, Vds = 40В, Id = 3.6А, RDS(on) = 44мОм тип / 56мОм макс (@VGS=10В, ID=3.6А)
- Затвор ← MAX1847 GATE
- Исток → R205 (0.1Ом) → PGND (датчик тока для CS MAX1847)
- Сток → T201 pin 4 (конец первичной обмотки)

**Паразитные эффекты IRLML0040 при 252 кГц:**

| Эффект | Параметр | Последствие | Решение |
|--------|----------|-------------|---------|
| Выброс на Drain при выключении | L_lk=0.4мкГн + C_oss=32пФ → звон ~1.4МГц | V_drain превышает 40В (Vds макс) | Снаббер BAS516+C208+2×BZT52C18, клампит до 36В; запас 4В |
| Эффект Миллера (C_gd ≈ 16пФ) | При нарастании V_drain ток через C_gd подзаряжает затвор обратно | Замедляет выключение, увеличивает потери переключения | R206=10Ом ограничивает скорость и контролирует dV/dt |
| Потери в C_oss | ½×32пФ×36²×252кГц ≈ 5мВт | Нагрев транзистора | Пренебрежимо мал |
| Осцилляция затвора | Паразитная индуктивность трасс PCB + C_gd → ВЧ-звон на Gate | Ложные включения, нестабильность | R206=10Ом давит осцилляцию; минимальная длина трассы Gate |

**Снаббер — детальная работа:**

При выключении Q201 ток первичной обмотки (через L_lk) не может оборваться мгновенно. V_drain начинает расти:
1. BAS516 открывается (Анод = Drain, потенциал растёт выше уровня катода)
2. Ток втекает в C208 (10нФ) — демпфирует звон, ограничивает скорость нарастания dV/dt
3. Когда V_drain достигает ~36В — открываются D202+D203 (2×BZT52C18 последовательно = 36В)
4. Избыточная энергия рассеивается в зенерах — V_drain зафиксировано на ~36В

Запас по Vds: 40В − 36В = **4В**. При длинных трассах к снабберу паразитная индуктивность PCB добавляет дополнительный выброс → снаббер должен быть **максимально близко к Drain Q201 и T201 pin4**.

### ATB322524-0110-T000 (TDK) — трансформатор
| Параметр | Значение |
|---|---|
| Производитель | TDK |
| Корпус | SMD 3.2×2.5×2.4 мм (1210/JIS) |
| Коэффициент трансформации | 1 : 10.2 |
| Индуктивность первичной | 7.0 ± 20% мкГн @ 100кГц |
| DCR первичной | 0.5 Ом макс |
| DCR вторичной | 60 Ом макс |
| Индуктивность рассеяния | 0.4 мкГн макс |
| Напряжение изоляции | 500 В rms |
| Номинальный ток | 0.7 А rms |

**Подключение:**
```
+5V_HV ─── T201 pin 1 (начало первичной обмотки, dot)
            T201 pin 4 (конец первичной) ─── IRLML0040 Drain ─── Анод D201 (BAS516)
            T201 pin 2 (начало вторичной, dot) ─── GND
            T201 pin 3 (конец вторичной) ─── C209 1нФ (первый конденсатор CW умножителя)
```

**Полярность вторичной обмотки:**
- Q201 включён: pin3 уходит в плюс (+94В), pin2 = GND → конденсаторы CW заряжаются
- Q201 выключён (флайбэк): pin3 уходит в минус → BAV23S умножителя проводят → −HV нарастает

### Снаббер первичного контура (RC + zener, по схеме референса)
```
T201 pin4 / Q201 Drain ─── D201 (BAS516) Анод
                            D201 Катод ─┬─ C208 10нФ ─── GND
                                        ├─ D202 (BZT52C18) Катод
                                             D202 Анод ─── D203 (BZT52C18) Катод
                                                           D203 Анод ─── GND
```
- **D201 BAS516** (SOD-523): быстрый диод, пропускает бросок Drain в снаббер
- **C208 10нФ**: демпфирует высокочастотный звон
- **D202 + D203** (два BZT52C18, SOD-123, последовательно): суммарный зенер 36В → ограничивают V_drain до ~36В
- Запас: 40В (Vds IRLML0040) − 36В = **4В** — достаточно при наличии C208

**Датчик тока Q201:**
```
Q201 Source ─┬─ CS (pin 13 MAX1847)
             ├─ R205 0.1Ом ─── GND
             └─ C205 100пФ ─── GND   (фильтр помех CS, параллельно R205)
```

**Рабочий цикл:**
- Vторичн_пик = Vвх × (Ns/Np) × D/(1-D) = 5В × 10.2 × 0.65/0.35 ≈ **94.6В** при D=0.65
- MAX1847 регулирует D в диапазоне, поддерживая заданное выходное напряжение

### Паразитные эффекты трансформатора при 252 кГц

**1. Индуктивность рассеяния (L_lk = 0.4 мкГн макс)**
Часть энергии, накопленной в первичке, не передаётся во вторичную обмотку — она сосредоточена в поле рассеяния. При выключении Q201 ток через L_lk не может оборваться мгновенно — возникает выброс напряжения на Drain. Именно L_lk является главной причиной применения снаббера (BAS516 + C208 + 2×BZT52C18). Рекомендация по PCB: минимальная площадь петли T201–Q201–C206/C204 для уменьшения паразитной индуктивности трасс, усиливающей выброс.

**2. Межобмоточная ёмкость (C_winding)**
Первичная и вторичная обмотки ёмкостно связаны. Быстрые перепады dV/dt на Drain при переключении (252 кГц) просачиваются через эту ёмкость прямо в высоковольтный выход CW умножителя, инжектируя высокочастотный шум. Это дополнительное обоснование необходимости 3-каскадного RC-фильтра после CW (раздел 5): он подавляет не только пульсации умножителя, но и этот кондуктивный шум.

**3. Потери в сердечнике**
При 252 кГц в феррите присутствуют два вида потерь:
- Гистерезисные (∝ f): перемагничивание сердечника за каждый цикл
- Вихревые токи (∝ f²): индуцированные токи в сердечнике

ATB322524 — ферритовый трансформатор, оптимизированный для диапазона 100 кГц–1 МГц. Частота 252 кГц находится в его нормальной рабочей точке.

**4. Скин-эффект в обмотках**
При 252 кГц глубина проникновения тока в медь:
```
δ = 66 / √252000 ≈ 0.13 мм
```
Ток вытесняется к поверхности проводника → эффективное AC-сопротивление обмоток превышает DCR (0.5 Ом первичка / 60 Ом вторичка). Вызывает небольшой дополнительный нагрев трансформатора, при токе 115 мА пренебрежимо мал.

**Итог:** все паразитные эффекты учтены в схемотехнике — снаббер (L_lk), RC-фильтр (C_winding), выбор компонента (потери в сердечнике), тепловой режим (скин-эффект).

### BAV23S в CW умножителе при 252 кГц

BAV23S используется как один диод (2 последовательных диода = 500В обратного напряжения, pin3 = NC).

- **Обратное напряжение на пару:** 2 × V_пик ≈ 375В < 500В ✓
- **Время обратного восстановления:** t_rr ≈ 50нс; доля периода: 50нс / 3970нс ≈ 1.3% — ток обратного восстановления пренебрежимо мал
- **Ток через конденсатор 1нФ:** I ≈ C × ΔV/Δt ≈ 90 мкА — потери на восстановление незначительны
- **Вывод:** диоды справляются с частотой 252 кГц без ограничений ✓

## 4. Вторичный контур: CW умножитель × 8

Вторичная обмотка T201 (pin3) подключается **напрямую** к первому конденсатору C209 CW умножителя — отдельный выпрямительный диод на вторичной стороне **отсутствует**. Функцию выпрямления выполняют диоды BAV23S внутри умножителя.

### Умножитель Кокрофта-Уолтона, 8 ступеней

**Конденсаторы:** 8 × **1нФ X7R 500В 0805** ±10%
Каждый конденсатор в ступени: V_ступени ≈ 94.6В, рабочее < 500В ✓

**Диоды:** 8 × **BAV23S** (SOT-23-3, два диода в корпусе, 250В, 200мА)
Итого 16 диодов = 8 полных ступеней умножителя.

**Расчёт (идеальный):**
```
V_out = 2 × n × V_пик = 2 × 8 × 94.6В ≈ 1514В → MAX1847 регулирует до ≤ 1500В ✓
```

Каждый диод BAV23S видит обратное напряжение 2 × 94.6В = 189В < 250В ✓

**Выходные накопительные конденсаторы:**
```
CW_out ─── C13a201─C13b201 (2× 0.1мкФ/1кВ в серии = 50нФ/2кВ) ─── GND
       └── C14a201─C14b201 (2× 0.1мкФ/1кВ в серии = 50нФ/2кВ) ─── GND
```
Тип: **C1812X104K102T (Holy Stone)** — 0.1мкФ X7R 1кВ, корпус 1812. По 2 штуки последовательно на каждую позицию (C13a201+C13b201 = одна позиция C13a201/C13b201).
⚠ Рабочее напряжение узла = полное выходное ВВ (~1500В). Только ВВ-исполнение ≥2кВ. Типовые 25В/100В керамика **недопустимы**.

## 5. RC-фильтры выхода

Три каскада RC-фильтрации для подавления пульсаций умножителя:

```
CW_out ─── R220 100кОм ─── node1 ─── R231 100кОм ─── node2 ─── R232 100кОм ───┬─── −HV_out
                    │                         │                           │      │
  C15a201─C15b201 2×0.1мкФ/1кВ  C16a201─C16b201 2×0.1мкФ/1кВ  C17a201─C17b201 2×0.1мкФ/1кВ  R233 10Ом
                    │                         │                           │      │
                   GND                       GND                         GND  C18a201─C18b201 2×220пФ C0G/1кВ = 110пФ/2кВ
                                                                               │
                                                                              GND
```
ВЧ фильтр R233+C18a201/C18b201 стоит **на выходе** (−HV_out), прямо перед нагрузкой (ФЭУ).
Напряжение на C18a201/C18b201 ≈ 1442В → 2× CC1206JKNPOCBN221 (220пФ C0G 1кВ 1206) последовательно = **110пФ / 2кВ**.
C0G обязателен: X7R при высоком DC-смещении теряет ёмкость, C0G стабилен.

Каждая позиция C15a/b201, C16a/b201, C17a/b201: **2× C1812X104K102T (Holy Stone) последовательно** (0.1мкФ / 1кВ X7R 1812):
- C_eff = 50нФ (0.1мкФ / 2), V_rating = 2кВ (1кВ × 2)
- Каждый конденсатор видит max 750В << 1кВ ✓
- DC-баланс без резисторов: для одинаковых керамических конденсаторов одной партии допустимо

**Постоянная времени:** RC = 100кОм × 50нФ = **5 мс** на каскад

**Падение напряжения на фильтре** (нагрузка ФЭУ 194 мкА):
194 мкА × 3 × 100кОм = **58В** суммарно при полной нагрузке.
MAX1847 компенсирует это в петле регулирования (closed-loop).

## 6. Управление: HV_CONTROL и обратная связь

MAX1847 содержит встроенный усилитель ошибки: регулирует скважность ШИМ так, чтобы **V_FB = 0В**. Внешний усилитель ошибки не нужен.

### Топология делителя FB

```
CW_out ── R207 ── R209 ── R210 ── R211 ── R212 ── R213 ── R214 ── R215 ── R216 ── R217 ── FB (pin 6)
          100М    100М    100М    100М    100М    100М    100М    100М    100М    100М       │
                              суммарно 1 ГОм                                           R208 2.2МОм
                                                                                    │
                                                                             [HV_CONTROL]
```

### Расчёт

Условие регулирования V_FB = 0В. На узле FB сходятся два тока:

```
V_CW / 1ГОм + V_CTRL / 2.2МОм = 0
V_HV = −V_CTRL × (1ГОм / 2.2МОм) = −V_CTRL × 454.5
```

| V_CTRL | V_HV |
|---|---|
| 0В | 0В |
| 3.3В | −1500В |

**HV_CONTROL:** PA8 (TIM1_CH1) PWM → RC-фильтр fc=160Гц → постоянное V_CTRL = 0…3.3В.

### Компоненты

| Ref | Номинал | Корпус | Назначение |
|---|---|---|---|
| R207, R209–R217 | 100МОм (×10) | 1206 | Верхнее плечо FB (суммарно 1ГОм) |
| R208 | 2.2МОм | 0402 | Нижнее плечо: задаёт коэффициент −454.5 |

Напряжение на каждом резисторе цепочки: 1500В / 10 = **150В << 200В** (макс. 1206) ✓

## 7. Мониторинг ВВ (HV_MONITOR)

Делитель: Vout− (0…−1500В) → 0…2.5В для MCU ADC (VREF = 2.5В).

```
−HV_out ─── R221–R230 (10 × 6.8МОм 1206 серия) ─── mon_node ─── R219 110кОм ─── катод TL431 (U202, 2.495В)
                                                        │
                                                HV_MONITOR ──── MCU ADC (PA0)
```

Буфер не нужен: R_th на mon_node = 68МОм ∥ 110кОм ≈ **110кОм**. MCU ADC справляется напрямую при времени выборки ≥ 247.5 цикла (требование: ~5.8мкс при ADC clock 42.5МГц).

**Расчёт делителя:**
- R221–R230 суммарно = 68 МОм, R219 = 110 кОм
- V_mon = Vref + (V_HV − Vref) × R219 / (R_chain + R219), где R_chain=68МОм, R219=110кОм
- V_HV = 0В:    V_mon = 2.5 + (0 − 2.5) × 110к/68.11М ≈ **2.496В** ✓ (≈2.5В)
- V_HV = −1500В: V_mon = 2.5 + (−1502.5) × 110к/68.11М ≈ **0.08В** ✓ (≈0В)
- Ток через делитель при 1500В: I = (1500В + 2.5В) / (68МОм + 110кОм) ≈ **22 мкА** (потребление HV цепью минимально)
- Каждый резистор 1206 (6.8МОм): 150В << 200В рабочих ✓

**Источник опорного 2.5В (Vref_2.5В) — TL431DBZ (U202):**
```
+5V_HV ─── R218 1.2кОм ─── Cathode (K)
                                  │
                            REF ──┤ (замкнут на K → V_K = 2.495В)
                                  │
                            C217 100нФ (на REF → GND)
                                 │
                                GND
TL431 Anode (A) ─── GND
```
- R218 задаёт ток катода: I_K = (5В − 2.5В) / 1.2кОм ≈ **2.1 мА** > 0.5 мА min ✓
- C217 (100нФ): стабилизирует TL431 против ВЧ-помех

**Прошивка:** канал PA0 (ADC2_IN1) — установить SampleTime = **247.5 cycles**.

## 8. Выход −ВВ на кабель

- Два пятачка с зазором ≥ 3 мм: Vout− и GNDA
- Припаять экранированный провод (изоляция ≥ 5 кВ, напр. ПВВ-1)
- Центральная жила → −HV → катод R1307 (через делитель)
- Оплётка → GNDA → земля делителя

## 9. Бюджет потребления

| Источник | Ток | От чего |
|---|---|---|
| MAX1847 + IRLML0040 (нагрузка ФЭУ 194 мкА × 1500В, η ≈ 55%) | ~110 мА | +5V_HV |

## 11. Требования к PCB

- Зазор −HV/GNDA от любых трасс: ≥ 3 мм (правило 1 мм/500В)
- Резисторы R221–R230 (6.8 МОм 1206): расположить в цепочку, зазор ≥ 1 мм между корпусами
- R207 (100 МОм): максимально далеко от влажных зон, покрыть лаком
- Первичный контур (T201, Q201): компактная петля переключения, минимальная площадь; C204(100нФ) рядом с MAX1847 IN
- CW умножитель: линейная цепочка от T201 pin3 к выходу, нарастающие зазоры к высоковольтному концу
- GNDA ↔ GND: соединять в одной точке (звёздная земля)
- TL431 (U202): близко к mon_node, C217 100нФ — максимально короткие трассы
- **Заливка силиконом** (вся плата после сборки и проверки):
  - Обеспечивает изоляцию ВВ узлов (−1500В) и влагозащиту
  - Заменяет лаковое покрытие — лак не нужен
  - Максимальная высота компонентов ограничена глубиной формы заливки
  - Перед заливкой: убедиться в отсутствии ERC/функциональных ошибок — демонтаж из силикона невозможен
  - Рекомендуемый силикон: ПТК Силагерм или аналог (диэлектрическая прочность ≥ 20 кВ/мм)
