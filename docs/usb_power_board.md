# Плата USB + питание

## Назначение
Нижняя плата стека спектрометра. Выполняет:
- Интерфейс USB-C ↔ UART (FT231XS)
- Формирование всех шин питания стека
- Индикация статуса (LED)

## Форм-фактор
- Круглая, **Ø 45 мм**, 2-слойная FR-4
- USB-C разъём **вертикальный, в центре платы** — кабель выходит с торца прибора
- Межплатный разъём 2×N (2.54 мм) по периметру — к плате МК сверху (размер при разводке)

---

## Архитектура питания

```
USB 5V (+4.75...5.25V, до 500 мА USB 2.0)
  │
  ├─── L101 47мкГн + 10мкФ тант. ─→ +5V_A  (для AD8000 +, 84дБ @ 300кГц)
  │
  ├─── U101: LP5907-3.3 ──────────→ +3.3V_A (аналог: VDDA, TL431DBZ)
  │
  ├─── U102: AMS1117-3.3 ─────────→ +3.3V_D (цифровое: STM32, флешка, FT231XS)
  │
  ├─── U104: TPS60400 + L102 47мкГн ─→ -5V_A  (для AD8000 −, 81дБ @ 250кГц)
  │
  └─── L103 100мкГн + C115 47мкФ + C116 100нФ ──→ +5V_HV (отфильтрованный USB 5В для MAX1847 на HV board)
```

**Бюджет тока из USB:**

| Шина | Потребитель | Ток |
|---|---|---|
| +5V_A | AD8000 (на плате делителя) + TPS60400 вход (charge pump) | ~58 мА |
| +3.3V_A | STM32 VDDA + TL431DBZ (U301) | 50 мА |
| +3.3V_D | STM32 ядро + флешка + FT231XS | 150 мА |
| −5V_A | генерируется TPS60400 из +5V_A; нагрузка AD8000 ~25 мА | — |
| +5V_HV | MAX1847 + flyback + CW-умножитель (HV board) | 112 мА |
| **Итого из USB** | | **~370 мА** ✓ (< 500 мА USB 2.0) |

## Компоненты

### U103 — FT231XQ (USB-UART, QFN-20)
- USB Full Speed → UART 600 кбит/с
- Питание: VCCIO от +3.3V_D; VCC от +3.3V_D; 3V3OUT только с bypass C18/C109
- CBUS0 → LED_USB (активность)
- Корпус QFN-20 с центральным exposed pad (EP к GND)

### U101 — LP5907MFX-3.3 (3.3V_A)
- LDO, 250 мА, SOT-23-5
- Шум: **6.8 мкВ rms** (10 Гц — 100 кГц) — критично для VDDA и TL431DBZ (U301)
- PSRR: 79 дБ @ 1 кГц
- Вход: +5V_USB, Выход: +3.3V_A
- Обвязка: C5/C103 1 мкФ X5R (вход), C6/C106 4.7 мкФ X5R + C7/C108 100 нФ X7R (выход)

### U102 — AMS1117-3.3 (3.3V_D, KiCad: AMS1117CD-3.3 TO-252, footprint→SOT-223)
- LDO, 800 мА, SOT-223
- Шум не критичен (цифровое питание)
- Вход: +5V_USB, Выход: +3.3V_D
- Обвязка: C8/C104 10мкФ тант. (вход), C9/C105 10мкФ тант. + C10/C107 100нФ (выход)

### U104 — TPS60400DBVT (-5V_A)
- Заряд-насос (charge pump), -5V из +5V_A
- Ток до 60 мА, f_sw = 250 кГц, SOT-23-5
- Обвязка: C11/C111 100нФ X7R (летающий CFLY, **X7R!**), C12/C110 100нФ X7R (bypass VIN, **X7R!**), C13/C112 10мкФ тант. (выход, **полярность обратная**: + к GND)
- **Фильтрация выхода:** L2/L102 47мкГн + C14/C113 10мкФ тант. (**+ к GND**) + C15/C114 100нФ X7R
  → f_c = 73 Гц → ослабление 250 кГц = 81 дБ, пульсации < 0.01 мВ на -5V_A

### J1 — USB-C разъём (вертикальный)
- **JAE DX07S024WJ3R400** — вертикальный SMD, монтаж в центре платы
- Кабель выходит перпендикулярно плате (вдоль оси стека, с торца прибора)
- ESD защита: PRTR5V0U2X (SOT-363, $0.30) — TVS на D+ D−

### J2 — Межплатный разъём 2×N (размер уточняется при разводке)
Передаёт все шины питания и сигналы UART вверх по стеку:

| 7 | UART_TX (FT231XS TXD → MCU RX) |
| 8 | UART_RX (FT231XS RXD ← MCU TX) |
| 9 | GND |
| 10 | LED_STATUS ← MCU |
| 11 | LED_ACTIVITY ← MCU |
| 12 | LED_HV ← MCU |
| 13 | LED_ERROR ← MCU |

## Индикация

Один светодиод, управляется от MCU через header. Состояния кодируются паттернами мигания в прошивке (готовность, набор спектра, ВВ, ошибка и т.д.).

| LED | Цвет | Сигнал | Назначение |
|---|---|---|---|
| D1 | Зелёный | LED | Статус устройства (паттерны мигания) |

+ D5 (синий, малый): LED_USB — от CBUS0 FT231XS (трафик USB).

Токоограничивающий резистор: 330 Ом (10 мА @ 3.3V).

## Стек плат (физический порядок)

```
[торец — USB-C кабель]
  USB/Power board       ← эта плата
  HV board              ← перехватывает HV_CONTROL/HV_MONITOR
  MCU board
  Divider board         ← ФЭУ R1307
[торец — детектор]
```

**Межплатные соединения:**
- USB board ↔ HV board: 2×N pin header (все шины питания + UART + LEDs + SWD)
- HV board ↔ MCU board: 2×N pin header (те же сигналы; HV_CONTROL/MONITOR
  терминируются на HV board, не проходят насквозь)
- HV wire: от HV board вдоль края стека к Divider board (в обход MCU board),
  экранированный провод 5 кВ изоляция

**Примечание:** HV board стоит в середине стека → HV_CONTROL и HV_MONITOR
в 2×12 header USB board направлены "вверх" и перехватываются HV board.
MCU board получает те же пины уже без HV_CONTROL/MONITOR (HV board их замыкает
на свою схему и не пробрасывает дальше).

## Открытые вопросы

1. **SWD для прошивки** — TagConnect TC2030 на плате МК (снять с торца через
   отверстие в корпусе) или вывести SWD пины на отдельный разъём USB board?
2. **Кварц FT231XS** — не нужен: FT231X работает с внутренним генератором
   (конфигурация через EEPROM). Экономим место и деньги. ✓
