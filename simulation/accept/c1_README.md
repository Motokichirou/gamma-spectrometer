# Приёмочный стенд C1 — просадка усиления делителя под нагрузкой

Эталон приёмки для находки **C1** (`review/REVIEW.md`). Методология — как у C2:
стенд заморожен, правится только DUT, порог задан ДО фикса, pass/fail выносит
холодная ревью-сессия. Если неверен стенд/порог — отдельная правка с обоснованием.

## Файлы
| Файл | Роль | Правится? |
|---|---|---|
| `dut_divider.cir` | DUT: делитель R401–414 + буферы Q401–403 + базовые R | **да — это и есть фикс** |
| `c1_bench.cir` | стенд: HV −1500 В + динодные токи (k=0/1) + .meas потенциалов | нет (эталон) |
| `c1_check.py` | пост-процессор: усиление Π(зазор)^0.7, сдвиг, T_C1 pass/fail | нет (эталон) |

Топология DUT сверена с `pcb/gamma_spectrometer.net` (R-значения и связи Q совпадают).

## Стимул (worst-case)
- HV = −1500 В; нагрузка = средние динодные токи при **30 кимп/с @ 2.614 МэВ**
  (DY8/7/6 = 19.1/4.8/1.2 мкА, δ~4; тёмновой 10 нА), из `detector_calcs.py`
  (I_anode_avg≈25.5 мкА, I_div/I_anode=7.6).
- Сравнение **k=0** (холостой) vs **k=1** (полная нагрузка).

## Метрика и порог
Усиление ФЭУ = **Π(зазор)^0.7** по множительным каскадам K→Dy1 … Dy7→Dy8
(Dy8→анод = сбор, исключён). **T_C1: |Δусиления(k=0→k=1)| ≤ 1 %.**

## Базовая точка «ДО ФИКСА» (текущий dut_divider.cir)
Воспроизводит независимое ревью:

| | значение |
|---|---|
| пер-зазорные просадки | −0.84 … −0.94 % |
| **СДВИГ УСИЛЕНИЯ** | **−4.71 %** |
| **T_C1** | **FAIL** |

Корень: коллекторы буферов Q401–403 запитаны от мягких узлов делителя
(Q401→DY7, Q402→DY8, Q403→midP) → средний динодный ток течёт обратно в цепочку
→ IR-просадка. Цель фикса: |Δ| ≤ 1 % (жёсткая шина коллекторов / стабилизация).
